# 基于SpringBoot+Vue的志愿服务联动帮扶综合服务平台

> 基于SpringBoot+Vue的志愿服务联动帮扶综合服务平台

## 目标用户
- 系统管理员 — 管理用户/角色/权限
- 帮扶责任人 — 管理贫困户档案、走访记录、帮扶措施、项目
- 贫困户 — 查看档案、发布需求、申请就业、报名培训
- 志愿者/社工 — 参与志愿活动、登记捐赠、组织培训

## 核心功能
- 用户与权限管理（RBAC）
- 贫困户档案管理（含家庭成员）
- 帮扶结对与措施跟踪
- 走访记录
- 扶贫项目管理与审核
- 困难需求与捐赠管理（资金/物资）
- 就业帮扶（岗位发布/申请）
- 技能培训（课程/报名/签到）
- 志愿服务（活动/签到/时长）
- 信息公开与数据统计看板

## 技术栈
- 后端: Java 1.8 + Spring Boot 2.7 + MyBatis-Plus + MySQL + Redis
- 前端: Vue 3 + TypeScript + Vite + Element Plus + ECharts + SCSS

## 设计风格
- 政府公益项目，需兼具庄重感与亲和力
- 主色: 扶贫绿 (#1E8E5E) 点缀: 温暖橙 (#F5A623)
- 目标: 对标 Linear 的高级感 — 简洁、干净、精致微交互
