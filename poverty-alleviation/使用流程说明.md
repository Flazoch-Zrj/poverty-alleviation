# 基于SpringBoot+Vue的志愿服务联动帮扶综合服务平台 — 完整使用流程

## 一、系统启动

```bash
# 1. 启动后端（IDEA 或命令行）
cd poverty-alleviation/backend
mvn spring-boot:run -Pdev

# 2. 启动前端（新开终端）
cd poverty-alleviation/frontend
npm run dev
```

访问 http://localhost:3000

## 二、管理员流程（admin / 123456）

### 2.1 用户管理
1. 登录后点击左侧 **系统管理 → 用户管理**
2. **新增用户**：点击橙色"新增用户"按钮 → 填写用户名、密码、姓名、手机号、角色 → 确认
3. 编辑/删除/启停用户：表格中对应操作按钮
4. **重置密码**：选中用户 → 点击"重置"（默认密码 123456）

### 2.2 角色管理
1. **系统管理 → 角色管理**
2. 新增/编辑角色名称、编码、描述

### 2.3 数据字典
1. **系统管理 → 数据字典**
2. 按字典类型查询（贫困原因/贫困等级/帮扶措施）
3. 新增/编辑字典项

### 2.4 工作台
1. 查看统计卡片：总贫困户、本月走访、已结对、待办事项
2. **精准帮扶项目**：6 张项目卡片 → 点击"查看详情"查看完整项目信息
3. 最近活动时间轴

## 三、帮扶责任人流程（zhangjianjun / 123456）

### 3.1 贫困户档案
1. 点击 **帮扶管理 → 贫困户档案**
2. **新增家庭**：填写户主姓名、身份证号、地址、家庭人口、年收入、致贫原因等
3. **查看成员**：点击家庭成员 → 可新增/删除成员
4. 支持按户主姓名/身份证号搜索、按状态筛选

### 3.2 走访记录
1. **帮扶管理 → 走访记录**
2. **新增走访**：选择走访家庭、填写日期、走访内容、下一步计划
3. 支持按家庭搜索和日期范围筛选

### 3.3 帮扶措施
1. **帮扶管理 → 帮扶措施**
2. **新增措施**：选择家庭、措施类型（产业/教育/医疗/住房/就业/金融）、填写内容、目标金额
3. 进度跟踪：通过滑块调整完成进度（0-100%）

### 3.4 项目管理
1. **帮扶管理 → 项目管理**
2. **新增项目**：填写项目名称、预算、描述
3. **审核流程**：提交后待管理员审核（通过/驳回）
4. 查看审核日志

## 四、贫困户流程（wangdashan / 123456）

### 4.1 我的档案
1. 查看个人基本信息、家庭情况、贫困级别
2. 查看家庭成员列表

### 4.2 我的需求
1. **发布需求**：选择需求类型（资金/物资/医疗/教育/技术）、填写标题、描述、目标金额
2. 跟踪需求状态：待解决/已对接/已完成

### 4.3 就业申请
1. 浏览招聘岗位列表（公司、薪资、工作地点）
2. **在线申请**：点击"申请"填写备注
3. 查看申请状态（申请中/已通过/已拒绝）

### 4.4 技能培训
1. 浏览培训课程列表（讲师、时间、地点、名额）
2. **立即报名**：名额未满时可报名
3. **我的报名**：查看已报名课程、签到状态
4. **签到**：参加培训后可签到

### 4.5 信息公开
1. 查看政策新闻、脱贫案例、通知公告、捐赠公示

## 五、志愿者流程（zhaohongxia / 123456）

### 5.1 志愿活动
1. **全部活动**：浏览所有志愿活动、报名/查看状态
2. **我的活动**：查看已报名的活动
3. **新增活动**：创建新的志愿活动
4. **签到/签退**：参加活动后签到，结束时签退，系统自动计算服务时长

### 5.2 捐赠管理
1. **资金捐赠**：登记捐款（捐赠方、金额、支付方式）
2. **物资捐赠**：登记物资（捐赠方、物资名称、数量、单位）
3. **统计卡片**：顶部显示捐赠总笔数、总金额、已到账、待确认

## 六、核心业务流程示例

### 完整的帮扶流程
```
管理员创建帮扶责任人账号
        ↓
帮扶责任人登录 → 新增贫困户档案
        ↓
建立结对关系 → 制定帮扶措施
        ↓
定期走访 → 填写走访记录
        ↓
贫困户发布需求 / 申请就业
        ↓
责任人对接资源 / 审核项目
        ↓
志愿者登记捐赠 / 组织活动
        ↓
脱贫验收 → 档案状态更新
```

### 培训报名流程
```
管理员/志愿者发布课程
        ↓
贫困户浏览课程列表
        ↓ 
报名（名额校验）
        ↓
参加培训 → 签到
        ↓
查看我的报名记录
```

### 项目审核流程
```
责任人提交项目申请
        ↓
管理员审核（通过/驳回）
        ↓
通过后进入实施阶段
        ↓
跟踪项目进度
        ↓
项目完成验收
```
